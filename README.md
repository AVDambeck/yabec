# Yet Another Basic/Expert Clone 
Yabec started as a way to codify the Old School Rennessance mix I was running at my table. It now is an open source project for others to work from. I'm very grateful for an amazing table, and for all the feedback they've given me.

# Direction
This is verision 3.0! The main game is for the most part here and settled in. Minions, travel,and combat will probobly continue to be tweaked. 

Moving forward, I intend to focus on adding strongholds and warfare, and making premade character sheets and other resources.

## Suggestions
I'm always open to suggestions. Feel free to open an issue on git, or reach out to me via email, discord, signal, matrix, irl, or any other platform you know me through. 

# Features
what kind of system is yabec? what can you expect?

- 4 basic classes, plus extras like druid, animalkin, and mystic
- b/x math, using decending armor class, and dodge roles
- relative distances and zones for theater of the mind
- hazard die
- player roles
