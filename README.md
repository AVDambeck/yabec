# Yet Another Basic/Expert Clone 
Yabec started as a way to codify the Old School Rennessance mix I was running at my table. It now is an open source project for others to work from.

# Direction
This branch is to facilitate a reoredering of the pages. I want to reduce page turning, and make it easier to find what you're looking for, especially on web. I'll be moving away from schemes and systems, and working on a more holistic framework. There will also be other changes related to balancing, reducing paperwork, speeding up play, and adjusting the scope.

## Suggestions
I'm always open to suggestions. Here's a list of things I'm activly looking for suggestions on:

- Disambiguate roles and rolls.

# Features
what kind of system is yabec? what can you expect?

- 4 basic classes, plus lots of extras
- b/x math
- decending armor class
- relative distances and zones
- mapping guides
- DM doesnt make attack rolls
- hazard die
- player roles 
- object oriented hooks for bite sized rules
