Yabec

Yet Another Basic/eXpert Clone is a open source game system. It's designed as an alternative to other rule sets for the dragon game.

The two primary books are:

The Guide Book: which will guide a new player to create a character and begin playing.

The Rule Book: which is complete set of game rules, as well as guidelines for changing them.
