Yabec

Yet Another Basic/Expert Clone is a open source game system. It's designed as an alternative to other rule sets for the dungeon game. This is version 2.7. Read it online at https://avdambeck.github.io/yabec/

Introduction

If you are familiar with B/X, OSE, or other OSR games, you are already largely familiar with Yabec. The differences you'll want to pay attention to are the use or fuzzy "zones" instead of exact measurement, class options which have been tweaked, and overland travel which is done as a point-crawl. Yabec tries to put die rolls in front of the players.
